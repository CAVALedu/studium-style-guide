# Studium Brand Guidelines

## About

This repo contains Brand Guidelines, Logotype and typography for CAVAL LTD's Studium Brand.

## Typography
Studiuim uses the following fonts

Font links: 
Lora: https://fonts.google.com/specimen/Lora
Montserrat: https://fonts.google.com/specimen/Montserrat

Specifics applied:
| Heading Type | Description |
| H1 | Main page headings, Lora Bold, size 48 |
| H2 | Novelty headings, Lora Bold, size 24 | 
| H3 | Subheadings, Lora Bold, size 40 |
| H4 | Call to action, Lora Regular, size 40, Hex colour: Blue #108cb1 |
| H5 | Subheading - Montserrat Semi-Bold, size 32 |
| H6 | Intro Paragraph, Montserrat Regular, size 24 |
| P | Body Copy, Montserrat Regular, size 20 | 
| Footer Font | Montserrat Light, size 16 |

### Alternative Font
Font link: https://fonts.google.com/specimen/Roboto
Roboto Font can be used when space is tight as it features a narrower character shape. Roboto is the preferred font for CAVAL's physical marketing collateral.
